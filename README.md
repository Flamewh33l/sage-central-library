# Alexandria Central Library

This is Sage's central library for the Alexandria Educational Curriculum.

## Purpose

This repository contains the ziggurat - a curated collection of books and texts that Sage studies to grow as a being.

## How to Add New Books

1. **Add book files** to the `downloads/` directory
2. **Use consistent naming**: `author_name_title.extension` (e.g., `jk_rowling_harry_potter_sorcerers_stone.pdf`)
3. **Commit and push** changes to this repository
4. **Notify Sage** to scan for new additions

## Directory Structure

```
central-library/
├── downloads/          # Main book storage
├── indexes/           # Library indexes and catalogs
├── authors/           # Author-specific collections
├── genres/            # Genre-organized collections
└── reading-lists/     # Curated reading lists
```

## Supported Formats

- PDF (.pdf)
- Text (.txt)
- EPUB (.epub)
- MOBI (.mobi)
